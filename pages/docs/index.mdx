# Introduction

StaticKit is the modern way to gather form submissions from your frontend _without writing a line of backend code_. It works great with React-based generators, like Next.js and Gatsby, and traditional HTML-based generators, like Jekyll and Hugo.

Here is a working example of a contact form implemented with the StaticKit HTML library:
