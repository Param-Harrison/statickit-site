# Stripe

Functions for building payment flows with your [Stripe](https://stripe.com) account.

| Type                              | Description                                                  |
| :-------------------------------- | :----------------------------------------------------------- |
| [`createCustomer`](/docs/functions/stripe/create-customer) | Creates a Stripe customer record    |
| [`createCharge`](/docs/functions/stripe/create-charge) | Creates a charge using the Charges API  |
| [`createSubscription`](/docs/functions/stripe/create-subscription) | Creates a subscription for a customer |
