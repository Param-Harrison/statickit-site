# equals

Validates that a function argument equals a given value.

## Configuration

This rule type accepts the following parameters:

| Key                        | Type                     | Default                        | Description                                                  |
| :------------------------- | :----------------------- | :----------------------------- | :----------------------------------------------------------- |
| `arg` *                    | string                   |                                | The name of the argument                                     |
| `value` *                  | string                   |                                | The value the argument must equal                            |

### Example

This validates that an argument called `message` is defined:

```json
{
  "type": "equals",
  "arg": "status",
  "value": "subscribed"
}
```

You can also validate nested arguments, using dots to denote the path:

```json
{
  "type": "equals",
  "arg": "fields.status",
  "value": "subscribed"
}
```
