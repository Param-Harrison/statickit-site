# Mailchimp

Perform tasks like adding or updating a contact in your [Mailchimp](https://mailchimp.com) account or tracking an event for automation. We have the following function types available for Mailchimp:

| Type                              | Description                                                  |
| :-------------------------------- | :----------------------------------------------------------- |
| [`addOrUpdateListMember`](/docs/functions/types/mailchimp/add-or-update-list-member) | Adds or updates a list member |
| [`createEvent`](/docs/functions/types/mailchimp/create-event) | Adds or updates a contact and tracks an event |
